# HKit Global API

This project was created using:

- Nest.js
- Mongo DB (Mongoose)
- Swagger (for documentation)
- Passport.js (access/refresh strategy)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

### Read docs [here](http://localhost:3000/api/docs)
